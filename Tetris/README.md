\# TetrisCore Engine Development



본 프로젝트는 상용 엔진의 도움 없이 C++만으로 시스템을 제어하고, 메모리 및 성능을 최적화하는 엔진 프로그래머의 역량을 기르기 위한 '테스트베드'입니다.



\## 주차별 마이크로 마일스톤



\### Week 1: Memory Tracking System (완료)

런타임 중 발생하는 모든 동적 할당을 감시하고, 프로그램 종료 시 메모리 누수를 100% 감지하는 시스템을 구축했습니다.

\- `new`, `delete`, `new\[]`, `delete\[]` 연산자 전역 오버로딩

\- 할당/해제 바이트 추적 및 누수(Leak) 감지 리포트 출력

\- 의도적 누수 상황에 대한 정확한 탐지 검증 완료



\### Week 2: Object Pooling System (예정)

\- 런타임 동적 할당 '0'을 목표로 하는 고정 크기 메모리 풀 설계



\### Week 3 \~ 6 (예정)

\- Vector/Matrix 수학 라이브러리 직접 구현

\- 고해상도 타이머(Delta Time) 및 비동기 게임 루프

\- 데이터 지향적 충돌 처리 코어 구현
